 AI-инструмент для планирования исследований биоэквивалентности (Кейс «ИФАРМА»)

Репозиторий команды для хакатона по разработке интеллектуального помощника, который автоматизирует создание синопсиса протокола исследования биоэквивалентности на основе мульти-агентной системы.

 Технологический стек

- Backend: Python + FastAPI
- Агенты: LangGraph / CrewAI
- База данных: PostgreSQL (SQLite для разработки)
- LLM: OpenAI / Claude API
- Документы: python-docx
- Frontend: Flutter
- Инфраструктура: Docker, GitHub Actions

 Структура репозитория

Краткое описание основных директорий:

- `backend/` — FastAPI приложение, агенты, работа с БД и шаблонами.
- `ui/` — Flutter клиент.
- `docs/` — диаграммы и дополнительная документация.
- `data/` — тестовые данные (gold standard, регуляторные правила).
